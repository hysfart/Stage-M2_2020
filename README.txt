# Stage de Mater 2 - NAC
# UFR des sciences
# Université de CAEN Normandie

Dans ce répertoire github, on placera les différents éléments constitutifs du stage qui s'est déroulé dans des conditions exceptionnels (en confinement durant 6 semaines au moins, lié à l'épidémie de covid-19 en europe occidentale).

Le répertoire s'organise comme : 

	-> Abinit-results : dans lequel on place les résultats de calculs 'ab-initio' faits à l'aide d'ABINIT, ainsi que les fichiers d'input, et dans la mesure du possible, les fichiers d'output liés.
